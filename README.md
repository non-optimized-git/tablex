# TableX 表析

问卷数据分析工具，解析 Excel 导出文件，生成三个并排表格（计数、百分比无高亮、百分比有高亮）。

## 功能特性

- **Excel 解析** - 支持 .xlsx、.xls 文件，拖拽或点击上传
- **智能分类** - 自动按关键词将题目分类到模块（用户画像、购车意向、智能驾驶等）
- **模块管理** - 可手动添加、删除、重命名模块，题目顺序始终保持不变
- **分组分析** - 支持按列分组，对比不同分组的数据差异
- **多选识别** - 自动识别多选题（英文逗号分隔的选项）
- **配置导入导出** - 支持保存和恢复分组、模块、题目选择等配置
- **MUJI 风格** - 简洁优雅的界面设计

## 快速开始

```bash
# 克隆项目
git clone https://github.com/your-username/tablex.git
cd tablex

# 启动本地服务器
python3 -m http.server 9000

# 访问页面
open http://localhost:9000/index.html
```

## 使用说明

### 1. 上传 Excel 文件
- 拖拽文件到上传区域，或点击选择文件
- 上传后显示文件名、行数、列数

### 2. 模块分组
- 题目自动按关键词分类到模块
- 点击模块名称可修改
- 点击 `×` 可删除模块（题目合并到相邻模块）
- 点击题目下方的 `+` 可在该题后添加新模块

### 3. 题目选择
- 勾选/取消勾选题目
- 使用"全选"/"全不选"快速操作
- 题目顺序始终保持不变

### 4. 分组设置
- 选择分组列（如：性别、年龄等）
- 点击选择要对比的分组值
- 支持拖拽排序已选分组

### 5. 下载 Excel
- 点击"下载 Excel"按钮
- 每个模块生成一个 Sheet
- 每个题目输出 3 个并排表格：
  - Table 1: 计数（次数）
  - Table 2: 百分比
  - Table 3: 百分比 + 高亮（绿=最高，红=最低）

## 技术栈

- **前端** - 纯 HTML/CSS/JavaScript，无构建步骤
- **Excel 解析** - SheetJS (xlsx.full.min.js)
- **Excel 导出** - ExcelJS (exceljs.min.js)
- **部署** - Vercel
